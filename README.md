# ppocrv3-rec-openvino

## install

```shell
git clone https://github.com/geoyee/ppocrv3-rec-openvino
cd ppocrv3-rec-openvino
mkdir -p build
cd build
cmake ..
make
```

## run

```shell
./openvino_test
```
